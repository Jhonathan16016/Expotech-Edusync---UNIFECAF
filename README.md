# EduSync

Sistema integrado de gestão e consulta educacional desenvolvido em Python e MySQL como projeto da Expotech UNIFECAF.

## Descrição

O Sistema Integrado de Gestão e Consulta Educacional é uma solução desenvolvida para auxiliar no gerenciamento e consulta de informações acadêmicas de forma prática e organizada. O sistema permite o cadastro, alteração, consulta e gerenciamento de dados relacionados ao ambiente educacional, centralizando informações importantes em uma única plataforma.

Com foco na eficiência e organização, a aplicação busca otimizar processos administrativos, reduzir o tempo gasto em consultas manuais e facilitar o acesso às informações por meio de uma interface simples e intuitiva. O projeto demonstra a aplicação de conceitos de programação, banco de dados e integração de sistemas para solucionar problemas comuns da gestão educacional.

## Requisitos para Execução

Para o funcionamento correto do sistema, é necessário possuir o Python instalado e as seguintes bibliotecas:

```bash
pip install mysql-connector-python
pip install tabulate
```

## Tecnologias Utilizadas

- Python
- MySQL
- mysql-connector-python
- Tabulate
- Getpass

## Observações

O sistema utiliza o módulo `getpass` para ocultar a digitação de senhas durante o login, proporcionando maior segurança ao usuário. Esse módulo faz parte da biblioteca padrão do Python e normalmente não requer instalação adicional.
